## About The Project

This project is a practice repository for implementing middleware in Node.js. It includes several middleware functions that can be used as examples for building your own middleware

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/omerozoglu/NodeJs-Middleware.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
